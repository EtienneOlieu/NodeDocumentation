Packages

A package is a package.json file which contains metadata and dependencies.

When we work with a node.js project we are going to be dependent on certain functionalities collected in packages. The most important one is express which is thge backbone of node.

In order to install express I will need to open my terminal and type: 

    
        npm install express

This initiaties the node package manager, an inbuilt package manager found in javascript, and tells it to install express. Once express is installed it needs to be initialized and instantiated. This is done in two ways, the second of which requires that the entire project is set to be a module, both should be done at the top of the app.js file. 

    
        const express = require("express");
        //
        import express from 'express';
        //either way we need to instantiate express
        const app = express();

The more direct way to do this is to create the package.json file yourself and then put this into it. Make sure to remember the quotation marks as it will not be correct json otherwise. 

    
        {"dependencies":{
            "express":"4.18.2"
        },
        }

Afterwards a simple command of "npm init" in the terminal will help you configure the rest.
