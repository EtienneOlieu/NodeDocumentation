Callback Functions

A function sent to another fucntion as an argument

In order to start talk about calbback functions, we need to touch on a couple of other concepts. First off, the anonymous function.

Anonymous functions

Sometimes we need some functionality in the code but it is not necessary to use that function again, therefore it can be useful to create an anonymous function. So instead of creating a function like so: 

    
        function random(min,max){
            return Math.floor((Math.random()*(max - min +1) -min))
        }

We can instead write the code like this:

    
        const randomAnonymousFunction = function random(min,max){
            return Math.floor((Math.random()*(max - min +1) -min))
        }
This can be taken a step further in order to save even more on key strokes by writing the same example as an arrow or lambda function instead: 
    
        const randomCompactArrowFunction = (min,max) =>  
        Math.floor((Math.random()*(max - min +1) -min))
        console.log(randomCompactArrowFunction(0,20));

The example above returns a random number between 0 and twenty.

Now for the callback

A callback function, as stated earlier, takes a function as an argument and then works on that function. In this example we are creating a function that can handle generic actions as text, returns the actions and a the name. Of course, this is a simple example and not really pertinent for anything else than an example. Although it does illustrate the capability of callback functions to allow for modular behaviour in the code

    
        function genericACtionPerformer(genericAction, genericName) {


            return genericAction(genericName)
         }
         
         const subtract = (name) => `${name} is subtracting.`
         const walk = (name) => `${name} is walking.`
         const read = (name) => `${name} is reading War and Peace.`
         
         console.log(genericACtionPerformer(subtract, "Tobias"))
            returns: 'Tobias is subtracting.'
         console.log(genericACtionPerformer(walk, "Nicolas"))
            returns: 'Nicolas is walking.'
